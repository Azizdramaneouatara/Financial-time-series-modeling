# Financial-time-series-modeling

This repository is about how to analyze financial time series using software such as R or Python. Most of the code shared in each chapter is a solution to the exercises created by Tsay in his book 'Analysis of financial time series'. I do provide addtional project which may specifically be about how to apply these knowledge to a frontier market such as the BRVM in West Africa . We have 12 chapters  thus 12 projects where we will discuss about building a decisional tools for investors which is going to help them understand the market in a better way.
